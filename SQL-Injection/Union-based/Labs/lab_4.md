# Лабораторная работа: SQL-инъекция с использованием оператора UNION, извлечение нескольких значений из одного столбца.

В этой лабораторной работе обнаружена уязвимость SQL-инъекции в фильтре категорий товаров. Результаты запроса возвращаются в ответе приложения, поэтому вы можете использовать атаку UNION для получения данных из других таблиц.

В базе данных содержится другая таблица, называемая users, со столбцами, называемыми usernameи password.

Для решения лабораторной работы выполните атаку SQL-инъекции UNION, которая позволит получить все имена пользователей и пароли, и используйте эту информацию для входа в систему под именем administratorпользователя.

# Решение:

Данная лабораторная похожа на лабораторную работу №3, запрос выглядит так:


'+UNION+SELECT+NULL,username||'~'||password+FROM+users--
